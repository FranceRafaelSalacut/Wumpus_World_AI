How to play

Build a Deterministic Finate Automata (DFA) Diagram to help your character navigate through the cave and defeat the goblin. There is a hidden among the level that you must acquire in order to slay the goblin.

The following are objects that can be seen in the dungeon

Player Character

    The one you control.
Goblin
    The evil that you must kill inside the caves.
Pit
    A bottomless pit. Falling may lead to death
Stench
    Inidicates that the goblin is nearby.
Breeze
    Indicates that the Pit is nearby.
Sword
    Randomly found inside the caves. Is used to kill the goblin

The following buttons can be used to make the DFA Diagram

Add State

    Creates a node to use for diagram.
Add Transition
    creates a line between two nodes. Click on the first node followed by the second to use.
Remove State
    Removes a node from the diagram.
Clear Canvas
    Resets the whole board.
Run
    Can run the Diagram and helps your character move.
Choose Start State
    Is used to pick a starting node.
Choose End State
    Is used to pick a final node.
